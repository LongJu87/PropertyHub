# PropertyHub
# Open Source Property Management Platform

一个现代化的物业管理平台，专为短租房东设计。PropertyHub使房东能够高效管理多个房源，同时为客人提供个性化仪表板，以查看预订详情、访问房源信息并预订城市周边的数字化体验。

## ✨ 主要功能

- **房东管理面板**：集中控制所有房源、预订和收入
- **客人仪表板**：查看预订详情、入住信息和房源文档
- **数字体验预订**：客人可预订当地旅游、餐饮和娱乐活动
- **自动化通信**：自动发送预订确认、入住指南和重要通知
- **支付集成**：安全的在线支付处理系统
- **多房源支持**：从单一界面管理无限数量的房源
- **移动响应式设计**：在桌面、平板和移动设备上完全适配

## 🚀 快速开始

### 先决条件

- Node.js (v18或更高版本)
- PostgreSQL (v14或更高版本)
- npm或yarn
